Welcome to the “Uncertainty Quantification in Inverse Problems with CUQIpy” mini-book!#

Note: this mini-book is under construction

This mini-book contains supporting material for the Technical University of Denmark (DTU) course: “02975: Introduction to uncertainty quantification for inverse problems”. It illustrates various concepts arising in Bayesian inverse problems with code examples. We use the library CUQIpy as a tool for demonstration.
